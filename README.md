## Introduction

Welcome to the Wafle Menu repository! This mod menu offers a range of functionalities for Gorilla Tag, enabling you to customize and elevate your gameplay.

## Features

- **Enhanced Gameplay:** Access various tools and features that augment your Gorilla Tag experience.
- **User-Friendly Interface:** A simple and intuitive menu design makes it easy to navigate and customize settings.
- **Customization:** Adjust settings to match your preferences and playstyle.
- **Regular Updates:** Expect ongoing updates and improvements to ensure compatibility and new features.

## Installation

Follow these steps to install Wafle Menu:

1. **Build Wafle Menu:** Run the `builder.bat` file to execute necessary commands and initiate the `exe` file. You can also directly run the `exe` as an administrator.
2. **Choose Installation Location:** Create a dedicated folder for Wafle Menu. Avoid the downloads folder to prevent conflicts.
3. **Windows Defender Compatibility:** Temporarily disable Windows Defender during installation as the builder modifies files.
4. **Wafle-Dependencies.dll:** Place this file in the `BepInEx/plugins` folder within your Gorilla Tag directory.

## Troubleshooting

Encountering issues? Try these steps:

- Double-check administrator privileges when running `builder.bat` or the `exe` file.
- Disable Windows Defender or any anti-virus software temporarily.
- Ensure your system meets the minimum requirements for the mod menu.
